# Mux

## Configure

| Tool               | Action                                            |
| ------------------ | ------------------------------------------------- |
| Visual Studio Code | Press ```F1``` then select ```CMake: Configure``` |
| Bash               | ```cmake -S . -B ./build```                       |

## Build

| Tool               | Action                      |
| ------------------ | --------------------------- |
| Visual Studio Code | Press ```F7```              |
| Bash               | ```cmake --build ./build``` |
